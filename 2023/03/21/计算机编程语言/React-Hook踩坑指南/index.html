React 是一个用于构建用户界面的库。感觉前端语言像刺猬一样，难以下手，而且是反直觉的，到处都是坑。

如何运行

感觉不像C/C++一样，直接将编译器配好，随便拿个记事本写代码就OK。

使用命令

  • 创建了一个名为 moz-todo-react 的文件夹
  • 为你的应用程序安装了一些 npm 包；
  • 写入 react 应用启动所需要的脚本文件；
  • 创建一系列结构化的子文件夹和文件，奠定应用程序的基础架构；
  • 如果你的电脑上安装了 git 的话，顺便帮你把 git 本地仓库也建好。

目录 src 是我们 React 应用源码存放的目录。

目录 public 包含了开发应用时浏览器会读取的文件。React 将目录 src 中的代码嵌入这个文件，从而浏览器才能运行此文件。 index.html中的有些内容关乎 create-react-app 的运作，因此除非你知道自己在做什么样的修改，否则不建议编辑这个文件。

文件 package.json 包含了 Node.js/npm 为了建立该应用程序所管理着的文件信息。
